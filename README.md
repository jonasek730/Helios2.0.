# Helios
Krátká lehká textová hra pro pobavení/zabavení. Tvým cílem je úspěšný únik z rozpadající se vesmírné lodi pomocí jednoduchých příkazů.
Celá hra funguje na jednoduchém while. Příkazy pro jednoduché ovládání jsou řešeny Command patternem. Data hry se načítají z Json souboru.
je velmi jednoduché si hru upravit podle vlastních představ s vlastním textem, názvy atd..  
## Instalace

1. Stáhni repozitář.
2. Otevři projekt ve svém IDE.
3. Spusť hlavní třídu "Main".

## Použití

- Spusť aplikaci a následuj instrukce na obrazovce.
- Komunikace probíhá přes konzoli.

## Požadavky

- Java 17+
- Maven
